

Java project--Least Recently Used cache.

Contains 3 java files.
2 classes LinkedListNode  and Cache defines data structures and methods needed to build system

Cache performs operations
	1. Add new value 
	2. Remove value
	3. Fetch value
	4. Reset Cache
	5. Delete least used values when capacity exceeds
	

Steps to run
 clone repo and run as java Application, file LRUCache.java
