Java project--Least Recently Used cache.

Folder src has 3 java files.
2 classes LinkedListNode  and Cache defines data structures and methods needed to build system.
LRUCache java file has void main method, run this file to run the program.

Cache performs following operations:
	1. Add new value 
	2. Remove value
	3. Fetch value
	4. Reset Cache
	5. Delete least used values when capacity exceeds
	
Steps to run:
	1.Clone repo
	2.Run as java Application (LRUCache.java)
	3. Programs take command line input, enter/press values as needed.
